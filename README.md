## What I've built

**[AI Career Copilot](https://github.com/Siddhhhh/ai-career-copilot)** — Full-stack AI SaaS. Upload a resume, paste a JD, get a semantic match score powered by Groq LLaMA 3.3 70B. Live resume editor with AI rewriting, interview prep with brutally honest AI feedback, role-specific coding challenges, multi-resume candidate ranking, GitHub profile verification. Next.js + FastAPI.

**[Market Intelligence Terminal](https://github.com/Siddhhhh/Market_Intelligence)** — Bloomberg-style financial platform. 3.7M+ rows of S&P 500 data in TimescaleDB. Two-model LLM architecture (Llama3 + Mistral) that answers financial questions grounded in real database evidence. Automated event detection across 36 years of market data.

**[AI Codebase Intelligence Engine](https://github.com/Siddhhhh/ai-code-intelligence)** — Analyzes entire GitHub repositories using local LLMs. Semantic vector search across every function and class. NetworkX dependency graphs with impact analysis. 108 tests. Zero API cost.

**[AI Forensics Toolkit](https://github.com/Siddhhhh/ai-forensics-toolkit)** — Detects AI-generated text and images using DistilBERT + ResNet18 feature extraction with RandomForest classifiers. Conservative thresholds to avoid overconfident misclassification.
